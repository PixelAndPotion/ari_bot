# ari_bot
A.R.I BOT is a python-based AI assistant with a GUI built using Tkinter. It provides live weather updates, cryptocurrency tracking, news headlines, movie search and details and has a built-in therapist mode. 
Tools include python, Tkinter (GUI), Requests (API handling).
This project was built to demonstrate GUI applications, work with API's and creating a python-based system for real-world cases.
How to run? python main.py
